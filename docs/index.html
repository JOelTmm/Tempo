Télécharger Tempo (Windows)

Application desktop gratuite — PC ou portable Windows uniquement.

Téléphone ou tablette : les fichiers .exe ne s’ouvrent pas sur Android ni iPhone. Utilisez un PC Windows (ou un Mac avec Windows en virtuel). Vous pouvez quand même rejoindre une partie en ligne depuis un PC après qu’un ami vous envoie le code salon.

Installer Tempo (.exe) Version portable

Je suis le développeur — publier une nouvelle version
  1. Dans le projet : npm run dist:win
  2. Sur GitHub : Releases → New release
  3. Glissez les fichiers .exe du dossier release/
  4. Nommez-les par ex. Tempo-Setup.exe et Tempo-Portable.exe

Jouer en ligne

  1. Téléchargez et installez Tempo sur chaque PC
  2. Créez un compte (email + mot de passe)
  3. Menu En ligne → Créer un salon
  4. Envoyez le code à vos amis → ils cliquent Rejoindre

Multijoueur Internet avec Supabase (configuré dans l'app). Sinon : salon local sur le même Wi‑Fi.
